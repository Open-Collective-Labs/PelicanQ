# Go SDK

## Installation

```bash
go get github.com/Open-Collective-Labs/PelicanQ/sdks/go/pelicanq
```

## Usage

```go
import "github.com/Open-Collective-Labs/PelicanQ/sdks/go/pelicanq"

client, err := pelicanq.Connect("127.0.0.1:7072")
// ...
```
